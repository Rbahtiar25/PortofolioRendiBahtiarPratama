# Analisis dan Pemantauan Kinerja Penjualan - Power BI Dashboard

Dashboard ini dibuat menggunakan **Microsoft Power BI** untuk membantu proses analisis dan pemantauan kinerja penjualan perusahaan secara lebih efektif.

## Fitur Utama & Informasi yang Ditampilkan

*   **Metrik Utama (Sales, Profit, & Quantity):** Menampilkan total penjualan (*Sales*), keuntungan (*Profit*), dan kuantitas penjualan (*Quantity*) yang dapat dianalisis berdasarkan tahun yang dipilih.
*   **Analisis Perbandingan Tahunan:** Selain menampilkan data pada tahun berjalan, dashboard juga menyediakan perbandingan dengan tahun sebelumnya sehingga pengguna dapat melihat perkembangan dan perubahan kinerja bisnis dari waktu ke waktu.
*   **Visualisasi Data Interaktif:** Berbagai visualisasi digunakan untuk menyajikan informasi secara jelas dan mudah dipahami, antara lain:
    *   **KPI Card** untuk metrik utama.
    *   **Grafik Tren Bulanan** untuk memantau pergerakan penjualan.
    *   **Peta Persebaran Wilayah** untuk melihat persebaran geografis.
    *   **Grafik Perbandingan Antar Region** untuk perbandingan regional.
    *   **Tabel Ringkasan** untuk detail data terstruktur.

## Manfaat & Penggunaan

Melalui dashboard ini, pengguna dapat:
1.  **Mengidentifikasi tren penjualan** secara berkala.
2.  **Mengevaluasi tingkat profitabilitas** perusahaan.
3.  **Membandingkan kontribusi** masing-masing wilayah terhadap kinerja perusahaan.

Visualisasi yang interaktif memungkinkan pengguna melakukan eksplorasi data secara lebih mendalam sesuai kebutuhan analisis. Dengan demikian, dashboard ini tidak hanya berfungsi sebagai media pelaporan, tetapi juga sebagai alat bantu dalam mendukung proses pengambilan keputusan yang lebih cepat dan berbasis data.
